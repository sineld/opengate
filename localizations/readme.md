# OpenGate Localizations

This folder contains community-contributed language files for OpenGate.

## How to Contribute

1. Fork this repository
2. Create a new file under `Localizations/` named with your language code (e.g. `de.json`, `fr.json`, `ja.json`)
3. Use `en.json` as a reference
4. Submit a Pull Request
